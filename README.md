🚀 Project Overview

This project is an interactive data analytics dashboard built using a Kaggle dataset, designed to transform raw data into meaningful business insights.

The goal is to explore patterns,trends and key performance indicators (KPIs) using visual analytics and provide a clear decision-making view through an interactive dashboard.
🎯 Objective of the Project

To demonstrate how raw Kaggle data can be converted into:

Interactive dashboards
Business insights
Visual storytelling reports

This project focuses on data visualization, analytics thinking, and real-world decision-making skills.

📂 Dataset Source
Dataset is taken from Kaggle
Data was cleaned, structured, and transformed before visualization
Original dataset contains multiple features such as:
Categorical variables (e.g., category, region, type)
Numerical values (e.g., sales, income, amount)

🛠️ Tools & Technologies Used
Microsoft Excel 
Kaggle Dataset
Data Cleaning & Transformation Techniques

📊 Dashboard Features

📌 Interactive filters for dynamic analysis
📈 KPI cards (Total, Average, Growth metrics)
📊 Category-wise comparison using bar charts
📉 Trend analysis using line charts 
🥧 Distribution analysis using pie charts


📈 Key Insights 
From the dataset analysis, the dashboard helps identify:

Top-performing categories or segments
Patterns and trends in data distribution
High and low performing areas
Relationships between key variables
Data-driven decision-making opportunities
